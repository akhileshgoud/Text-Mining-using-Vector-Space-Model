Text Mining using Vector Space Model



This program takes keywords as input terms that form a sad or congratulatory context, then looks up (searches) the word in the pre-defined training dataset to find their TF-IDF values. Later it calculates the similarity using cosine function (Vector Space Model) and predicts the context for the given terms based on this similarity measure.
